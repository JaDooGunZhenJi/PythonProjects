#  Pet Game

파이썬 클래스(Class)를 사용하여 만든 콘솔 기반 펫 키우기 게임임.

##  프로젝트 소개

사용자가 펫을 생성하고 다양한 활동을 하며 상태를 관리하는 게임이며, 또한 이 게임은 내 자신을 키우는 게임이라고 생각하고 만듦.
데이터를 파일에 저장하고 다시 불러올 수 있음.

## 주요 기능

- 새 게임 시작
- 저장된 게임 불러오기
- 밥 주기
- 놀기
- 여가 활동 하기
- 운동하기
- 상태 보기
- 내일로~
- 게임 저장
- 게임 불러오기
- 종료

## 사용 기술

- Python
- Class (객체지향 프로그래밍)
- File I/O
- Module 분리
- 예외 처리 (try / except)
